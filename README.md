# HR Management System

A beginner-friendly Human Resource Management System built with Flask for quick demo and submission.

## Features
- Add new employees
- View employee directory
- Track employee status

## Run locally
1. Install dependencies:
   pip install -r requirements.txt
2. Start the app:
   python src/app.py
3. Open http://127.0.0.1:5000
